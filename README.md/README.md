# GitHub Repo Explorer

A full-stack web application built with React and Express.

## Features

* Search GitHub users
* View profile details
* View repositories
* Sort repositories by Stars, Name, and Last Updated
* Recent Searches
* Loading Indicator
* Error Handling
* 60-second Cache

## Tech Stack

### Frontend

* React
* Vite
* Axios

### Backend

* Node.js
* Express
* Axios
